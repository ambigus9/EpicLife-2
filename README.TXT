------------------------------------------------------------------------
Este proyecto contienen la arquitectura y el funcionamiento de un pequeño videojuego
que se planea inplementar para android.
------------------------------------------------------------------------

PROJECT TITLE: EpicLife 
PURPOSE OF PROJECT:El proposito del proyecto es desarrollar la logica y la interacción entre clases que 
es necesaria para todas las interacciones del jugador con el videojuego. posteriormente el juego se migrara a
la plataforma de android, en donde se podra concentrar en el apartado visual. 
VERSION or DATE: 08-11-16
HOW TO START THIS PROJECT:El proyecto hace parte del desarrollo de la asignatura de principios y practicas de 
programación orientada a objetos. Las dos primeras entregas se basaban en el diseño y este es el primer 
prototipo. 
AUTHORS: Henry Jiménez y Miguel Plazas.
